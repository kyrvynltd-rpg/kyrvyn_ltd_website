# Security Policy

## Supported Versions

Only the `main` branch is supported.

## Reporting a Vulnerability

- Do not open public issues for security reports.
- Email: security@kyrvyn.co.uk
- Include steps to reproduce, impact assessment, and any suggested remediation.

## Dependency Vulnerabilities

- Automated scanning is enabled via Dependabot and CI auditing.
- High/Critical vulnerabilities should be addressed within 72 hours, or documented in the remediation report with an owner and target date.
